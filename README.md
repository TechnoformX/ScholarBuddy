## ðŸ”§ Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn
- Python 3 (if using Flask)
- OpenAI API Key (for chatbot)

### Installation Steps

```bash
git clone https://github.com/yourusername/scholarbuddy.git
cd scholarbuddy
npm install
npm start
```

_For Flask backend (optional):_

```bash
cd backend
pip install -r requirements.txt
python app.py
```
